WeatherBar의 체감온도를 기본 파란색으로 유지하되, 31℃ 이상부터 연노랑 → 빨강으로 점진적으로 변하는 색상을 적용합니다.

### 변경 파일
- `src/components/WeatherBar.tsx`

### 구현 내용

1. 체감온도 숫자 추출
   - 기존 `current.chill.replace(/체감\(|\)|℃/g, "")` 방식을 그대로 활용해 숫자값을 얻습니다.
   - 파싱된 값을 `Number()`로 변환해 구간 판별에 사용합니다.

2. 구간별 색상 함수 추가
   - 31℃ 미만: 파란색 (`text-blue-700`, 현재와 동일)
   - 31℃ ~ 32℃: 연노랑 (`text-yellow-400`)
   - 33℃ ~ 34℃: 노랑/주황 (`text-yellow-500`)
   - 35℃ ~ 37℃: 주황 (`text-orange-500`)
   - 38℃ 이상: 빨강 (`text-red-600`)

   Tailwind의 의미 클래스를 사용하며, 동적으로 클래스 문자열을 반환하는 헬퍼 함수를 추가합니다.

3. 체감온도 렌더링 수정
   - "체감" 레이블은 기존처럼 작은 파란색(`text-blue-500`)으로 유지합니다.
   - 체감온도 숫자 부분에만 구간별 색상 클래스를 동적으로 적용합니다.

4. 검증
   - `bun run build`로 TypeScript 및 Tailwind 클래스 오류 여부 확인
   - 미리보기에서 31℃ 미만 / 31℃ / 33℃ / 35℃ / 38℃ 구간에서 색상 변화 확인
